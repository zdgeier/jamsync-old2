It's time for a better version control system.

Version control is critically important to every developer's workflow. It allows us to backup our code, restore files when things go wrong, and ensure consitency across our codebase. However, the tools (Git, Perforce, Subversion, etc.) we use today to do these tasks are often painful to learn, slow, and require lots of manual effort. Additionally, these systems were all created in the mid-2000's when network and storage systems were very different. We can do better.

Jamsync is a version control system for solo developers designed from the ground-up to be easy-to-use and simple. Instead of having to go through several steps every time you want to make a change to your project, Jamsync only has a single command -- jam sync -- to start tracking your changes. We store every change you make to your project so you can always access it later.
